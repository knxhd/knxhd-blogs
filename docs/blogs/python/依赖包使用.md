## FastAPI

 ### 安装

1. 使用pip或conda安装 FastAPI

```shell
pip install FastAPI
# 或
# conda install fastAPI
```

2. 安装`Uvicron`，用于运行FastAPI定义的接口，相当于是一个web容器

```shell
pip install uvicron
```

### 启动项目

1. 示例项目，文件名为 test.py

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/")
def read_root():
    return {"Hello": "World"}
```

2. 启动项目

```shell
uvicorn test:app --port 8080 --reload

# --port 可选项，如果没有，则默认8000
```
