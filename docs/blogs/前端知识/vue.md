## npm

1. npm使用淘宝镜像

```shell
npm i node-sass --sass_binary_site=https://npm.taobao.org/mirrors/node-sass/
```

2. 设置淘宝镜像

```shell
npm config set registry https://registry.npm.taobao.org
```
