## 提示词工程

### 专业名词

#### 零样本提示

>- 即 对于训练后的大模型，提示词中不包含任何示例说明，例如
>
>  - 提示
>
>  ```java
>  将文本分类为中性，负面或积极。
>  
>  文本：我认为假期还好。
>  情感：
>  ```
>
>  - 输出结果
>
>  ```
>  中性
>  ```

#### 少样本提示

>- 在提示词中添加少量的示例，便于大模型按照示例的格式输出最终的结果，例如
>
>  1. 提示
>
>  ````
>  将文本分类为中性，负面或积极。
>  
>  文本：我认为假期还好。 情感：积极
>  文本：我今天心情不好 
>  情感：
>  ````
>
>  2. 输出
>
>  ```
>  负面
>  ```
