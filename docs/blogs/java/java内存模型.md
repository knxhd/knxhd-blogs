## Java内存模型的抽象

> 在Java中，所有的实例域、静态域和数组元素都存储在堆内存中。堆内存中变量在各个线程中可以共享。而局部变量、方法参数和异常参数等，不会在线程之间共享，他们不会存在内存可见性问题，也不会受内存模型的影响。

>线程之间的通信是空过Java内存模型来控制的。线程A和线程B共享主内存中的变量，线程A读取内存中的数据到主线程中，然后修改数据后，再将数据写入到主线程中，线程B通过主线程读取到修改后的变量，此过程称之为线程之间的通信。
